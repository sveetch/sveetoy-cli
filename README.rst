===================
Sveetoy Commandline
===================

Commandline tools around `Sveetoy`_ Sass library.

Links
*****

* Download its `PyPi package <http://pypi.python.org/pypi/sveetoy-cli>`_;
* Clone it on its `Github repository <https://github.com/sveetch/sveetoy-cli>`_;

Dependancies
************

* ``python-venv`` package;
* `six`_;
* `click`_ >=5.1, <6.0;
* `colorama`_;
* `colorlog`_;
* `colour`_;
